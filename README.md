# 🚀 AI-Powered Fake Email Detection System

## 📌 Overview

This project is a Machine Learning-based web application that detects whether an email is **Fraudulent (Phishing)** or **Legitimate (Safe)**.

It uses Natural Language Processing (NLP) techniques to analyze email content and classify it using a trained ML model.

---

## ⚡ Features

- 🔍 Detects phishing emails instantly
- 🤖 Machine Learning model (Logistic Regression)
- 🧠 NLP-based text preprocessing
- 📊 Confidence score visualization (Chart.js)
- 🌐 Modern UI using Tailwind CSS
- ⚡ Fast API with Flask

---

## 🛠️ Tech Stack

- Python
- Flask
- Scikit-learn
- NLTK
- HTML, CSS (Tailwind)
- JavaScript (Chart.js)
